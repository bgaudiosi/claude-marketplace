# ben-dev

Claude Code Plugins by bgaudiosi

## Installation

Add the marketplace to Claude Code:

```
/plugin marketplace add bgaudiosi/claude-marketplace
```

Install a specific plugin:

```
/plugin install example-plugin@ben-dev
```

## Plugins

- **example-plugin**: Example plugin demonstrating command structure

## Development

Run the linter to validate plugin structure:

```bash
make lint
```

Update plugin documentation and website:

```bash
make update
```

Visit the [documentation site](https://bgaudiosi.github.io/claude-marketplace) for more information.
